# To-Do List 应用

这是一个使用纯HTML、CSS和JavaScript开发的简单而强大的待办事项应用。

## 功能特性

- ✅ 添加新的待办任务
- ✅ 标记任务为已完成/未完成
- ✅ 删除任务
- ✅ 编辑现有任务（双击任务文本）
- ✅ 过滤任务（全部/未完成/已完成）
- ✅ 清除已完成的任务
- ✅ 本地存储支持（刷新页面后数据不会丢失）
- ✅ 响应式设计，支持移动设备
- ✅ 优雅的动画效果
- ✅ 空状态提示

## 如何使用

1. 在输入框中输入任务内容
2. 点击「添加」按钮或按回车键添加任务
3. 点击任务前的复选框标记任务完成状态
4. 双击任务文本可以编辑任务内容
5. 悬停在任务上点击「×」按钮删除任务
6. 使用顶部的过滤器切换不同类型的任务视图
7. 点击「清除已完成」按钮批量删除已完成的任务

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 交互逻辑
- **localStorage** - 本地数据存储

## 开发说明

1. 克隆或下载此仓库
2. 直接在浏览器中打开 `index.html` 文件即可使用
3. 无需任何构建工具或依赖项
